IMSI Catching Attack
==========================================================

Implementation (gNB)
--------------------------
Create a dummy gNB:

- Configure the gNB to have a higher signal strength than the other
- Configure the dummy gNB to send IMSI and other sensitive data to a server

Mitigation
------------
- Verify gNB identity
- Use encryption

Attack Metrics
----------------
- Captured IMSIs
- Disconnected UEs
- Disrupted network function
